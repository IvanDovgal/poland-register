Help

\r\n
\r\n
\r\n

\r\n 1.Schedule an appointment:To schedule an appointment use Schedule an appointment link on the left menu. Scheduling an appointment involves following steps:\r\n

\r\n
    \r\n
  • a.\tSelect Mission (option available only for CC User. For Individual /TA’s this option is disabled.)
    • \r\n
  • b.\tSelect Country (option available only for CC User. For Individual /TA’s this option is disabled.)
    • \r\n
  • c.\tSelect location : Center can be visa application center (VAC), Embassy
    • \r\n
  • d.\tSelect Visa Category
    • \r\n
  • e.\tClick on “Continue” button to go to Applicant List Page.
    • \r\n
  • f.\tApplicant list: Initially applicant list will be empty. Login User need to add applicant/s before he/she proceed with appointment date selection.
    • \r\n
  • g.\tDate & time slot selection: After adding applicant/s, on click of continue, user will be taken to final calendar page where user will see appointment dates (in green) and holidays (in red).
    • \r\n
  • h.\tUser can select the date and time slot and confirm.
    • \r\n
  • i.\tAppointment confirmation: Click on “Confirm” button in the final calendar page
    • \r\n
  • j.\tAn email & SMS will be triggered to the applicant/s after appointment is confirmed.
    • \r\n
\r\n

\r\n 2.\tReschedule an appointment: To reschedule an appointment, user need to have the reference number with valid future appointment. User can click on “reschedule an appointment” link.\r\n

\r\n
    \r\n
  • a.\tEnter the reference number and click on submit to get the applicant/s details
    • \r\n
  • b.\tUser can select the applicant/s to reschedule and click on continue to proceed to final calendar to select date & time
    • \r\n
  • c.\tUpon selection date & time, can confirm the appointment by clicking confirm button.
    • \r\n
  • d.\tAppointment confirmation message will be displayed
    • \r\n
  • e.\tAn email & SMS will be triggered to the applicant/s
    • \r\n
\r\n

\r\n 3.\tCancel an Appointment: To cancel an appointment, user need to have the reference number with valid future appointment. User can click on “Cancel an appointment” link.\r\n

\r\n
    \r\n
  • a.\tEnter the reference number and click on submit to get the applicant/s details
    • \r\n
  • b.\tUser can select the applicant/s to cancel and click on continue.
    • \r\n
  • c.\tAppointment cancellation confirmation message will be displayed
    • \r\n
  • d.\tAn email & SMS will be triggered to the applicant/s
    • \r\n
\r\n

\r\n 4.\tPrint or Email Appointment Letter: To print an appointment letter, user need to have the reference number with valid future appointment. User can click on “Print an appointment” link.\r\n

\r\n
    \r\n
  • a.\tEnter the reference number and click on submit to get the applicant/s details
    • \r\n
  • b.\tUser can select the applicant/s to print and click on “Download appointment letter” button.
    • \r\n
  • c.\tAn email & SMS will be triggered to the applicant
    • \r\n\r\n
\r\n

\r\n 5.\tRetrieve Incomplete Appointment: To retrieve an incomplete appointment, user need to have the reference number. User can click on “Retrieve Incomplete appointment” link.\r\n

\r\n
    \r\n
  • d.\tEnter the reference number and click on submit to get the applicant/s details
    • \r\n
  • e.\tLogin User can also add applicant/s before he/she proceed with appointment date selection.
    • \r\n
  • f.\tUser can select the applicant/s to schedule and click on continue to proceed to final calendar to select date & time
    • \r\n
  • g.\tUpon selection date & time, can confirm the appointment by clicking confirm button.
    • \r\n
  • h.\tAppointment confirmation message will be displayed
    • \r\n
  • i.\tAn email & SMS will be triggered to the applicant/s
    • \r\n\r\n
